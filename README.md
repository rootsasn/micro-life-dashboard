# Micro Life Dashboard

A brutally simple, offline-first daily essentials app.

## Structure

```
/web-pwa          - React + Vite + Tailwind PWA
/flutter-app      - Flutter mobile app
```

## Quick Start

### Web PWA
```bash
cd web-pwa
npm install
npm run dev
```

### Flutter App
```bash
cd flutter-app
flutter pub get
flutter run
```

## Features (MVP)
- 5 max tasks (checkbox + text)
- 5 max quick notes (text + date)
- 2 max reminders (text + timeframe)
- Offline-first with local storage
- <1s load time
- Clean, minimal UI
